# MAU 直驱 EC 插入式离心风机设计规格

## 目标

将 MAU-01 当前类似轴流螺旋桨的平面风机，重建为符合现代 MAU/AHU 常见结构的单台直驱 EC 插入式离心风机（plug/plenum fan）。风机必须在设备剖视中容易识别，同时正确表达轴向吸入、叶轮径向做功和静压箱汇流的工作过程。

本次只修改 MAU 风机段及其局部气流表达，不改动其他设备、公开嵌入 API、八段工艺顺序、慢速主气流、系统全景和 `original/` 快照。

## 结构依据

造型以 ebm-papst RadiPac、ZIEHL-ABEGG ZAbluefin 和 Greenheck plenum fan 等产品族的共同工程结构为校准依据，不复制任何厂家的专有外观：

- 无蜗壳，叶轮直接向风机静压箱排风。
- 空气沿叶轮轴线进入喇叭形进风锥，在后弯离心叶片中转为径向流动。
- 叶轮由前侧环形轮盖、后侧轮盘和有明显轴向深度的后弯叶片组成，不是单层平面叶片。
- 直驱 EC 电机位于叶轮下游侧，通过中心轴与后轮盘连接。
- 风机、电机和进风板由独立支撑框架及减振件承载。

当前模型的主要错误是风机旋转轴沿场景 Z 轴，而 MAU 主气流沿 X 轴，导致气流从叶轮侧面穿过。新模型必须将风机局部旋转轴映射到场景 X 轴。

## 选定方案

### 风机类型

采用一台直驱 EC 插入式离心风机。以当前演示工况 `18,200 m³/h` 为展示尺度，不改成传统皮带蜗壳风机，也不增加 2×2 风机墙。

### 组件分层

风机组保持 `supply-fan` 根节点，并包含以下可测试组件：

- `supply-fan-inlet-panel`：带圆形开口的方形进风隔板。
- `supply-fan-inlet-cone`：喇叭形进风锥，入口位于气流上游侧。
- `supply-fan-front-shroud`：环形前轮盖，遮住叶片根部并形成叶轮眼。
- `supply-fan-rear-disc`：后轮盘，连接叶片与电机轴。
- `supply-fan-rotor`：旋转组，局部 Z 轴经根节点旋转后对应世界 X 轴。
- 七片 `supply-fan-blade-*`：有轴向深度的后弯离心叶片，使用挤出或参数化曲面几何。
- `supply-fan-drive-motor`：位于下游侧的 EC 电机，包含壳体、端盖、散热筋和中心轴。
- `supply-fan-support-frame`：连接进风板、电机和底座的焊接框架。
- 四个 `supply-fan-isolator-*`：位于框架底部的减振件。

前方不再显示遮挡进风口的实心中心轮毂。叶轮眼保持深色开放，电机通过后轮盘和局部剖视可见。

### 朝向与观察

MAU 空气从世界坐标负 X 流向正 X。风机入口朝负 X，电机位于正 X。风机根节点绕 Y 轴旋转 `-π/2`，使现有局部 Z 轴旋转动画映射到世界 X 轴。

默认 MAU 镜头保持检修侧观察，但增加轻微上游方向偏移，形成侧前方三分之四视角，使进风锥、叶轮深度、电机和支撑框架可以同时辨认。移动端仍以整机完整入镜为优先。

## 气流动画

主气流仍以约 11–13 秒穿越整台 MAU，不提高整体速度。风机段增加独立局部示踪粒子，分成三个连续阶段：

1. 粒子沿 X 轴收敛进入进风锥和叶轮眼。
2. 粒子经过叶轮后沿 Y/Z 方向径向扩散，表现离心做功。
3. 粒子在静压箱内重新转向正 X，进入消声段。

局部示踪使用现有送风青色，不添加新的颜色体系。风机叶轮继续随运行状态旋转；`prefers-reduced-motion` 下保持现有减速策略。

## 材质与可读性

- 进风隔板和支撑架使用冷钢灰，避免纯白发光圆环。
- 叶轮使用深青金属，边缘仅保留低强度冷色高光。
- 前轮盖和后轮盘使用不同明度，明确叶轮轴向深度。
- 电机使用工业蓝灰并显示散热筋，不使用高饱和装饰色。
- 透明或剖切材质只服务于内部结构识别，不把风机做成发光装饰物。

## 兼容边界

- 保持 `buildMau(options)` 返回结构和 `group.userData.animation` 契约。
- 保持 `supply-fan-rotor` 名称，现有统一动画更新继续驱动它。
- 保持 `MAU-01` 的设备 ID、工艺段数据、HUD、嵌入 API 和镜头控制接口。
- 不修改 `original/`、冷水机组、水泵、冷却塔和系统管网模型。

## 测试与验收

自动测试必须先失败再实现，并验证：

- 风机包含进风隔板、进风锥、前轮盖、后轮盘、EC 电机、支撑框架和四个减振件。
- 叶轮恰好包含七片后弯叶片，每片具有非零轴向深度。
- 风机轴线与世界 X 轴一致，入口位于上游、电机位于下游。
- 局部风机气流路径至少覆盖轴向吸入、径向扩散和下游汇流三个阶段。
- 原有 MAU 分段、慢速主气流、设备动画和公开 API 测试继续通过。

真实浏览器在 1440×900 和 390×844 下验收：

- 风机不再呈现轴流花瓣或螺旋桨轮廓。
- 能同时辨认进风锥、深筒后弯叶轮、后置电机和支撑框架。
- 气流进入叶轮后发生径向转向，而不是直接水平穿模。
- MAU 全部分段仍完整入镜，无文字重叠、页面溢出或控制台错误。
- WebGL 画布非空且持续动画。
